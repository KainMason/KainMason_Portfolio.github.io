CS-499 Progress

Milestone Two – Software Design & Engineering (Travlr Getaways)

For Milestone Two, I enhanced my Travlr Getaways project to improve structure, maintainability, and clarity so it behaves more like a production-ready MEAN application instead of a one-off class assignment.

  • Refactored backend structure: separated routes, controllers, and config files.
  • Improved error handling: standardized API responses and HTTP status codes.
  • Cleaned MongoDB queries: removed redundancy and standardized query patterns.
  • Enhanced documentation: expanded the README and added an architecture overview.
  • UI consistency updates: tightened layout and formatting on customer-facing pages.

Milestone Three – Algorithms & Data Structures (3D Desk Scene)

For Milestone Three, I focused on improving the algorithms and data structures behind my C++ / OpenGL 3D Desk Scene so it is cleaner, safer, and easier to extend.

  • Geometry algorithm refactoring: consolidated repeated shape generation into reusable functions.
  • Stronger data structures: replaced raw arrays with std::vector and grouped vertex, normal, and texture data into structured containers.
  • Transformation pipeline cleanup: clarified the order of scaling → rotation → translation and improved model/view/projection logic.
  • Improved documentation: added comments explaining geometry-building steps and algorithm intent.
  • Stability improvements: added basic bounds checks, removed unused code, and fixed minor rendering issues.

Across these artifacts, I focus on writing clear, maintainable code, documenting my decisions, and building solutions that would be usable in real environments, not just in the classroom.
